Gặp rắc rối về đăng nhập

Nếu bạn đang gặp khó khăn khi đăng nhập, hãy kiểm tra để chắc chắn rằng cookie vẫn đang hoạt động trong trình duyệt của bạn. Trang web này sử dụng cookie để lưu giữ tên đăng nhập và mật khẩu của bạn (dưới dạng mã hóa) ở máy tính của bạn khi trong suốt quá trình bạn đăng nhập.

Để bật cookie, hãy làm theo các hướng dẫn dưới đây dành cho phiên bản trình duyệt mà bạn đang sử dụng.

Binary (C.R.) S.A.

Chrome 51+

  • Go to "Chrome Menu (Customize and control Google Chrome)" dropdown and select Settings.
  • Nhấp vào Hiển thị cài đặt nâng cao.
  • Nhấp vào nút cài đặt nội dung trong đơn bảo mật.
  • Select Allow local data to be set (recommended).
  • Click vào nút xong để lưu cài đặt của bạn.
Binary (IOM) Ltd

Mozilla Firefox 47+

  • Open menu and select Preferences.
  • Nhấp vào bảo mật tab.
  • Set "Firefox will" to "Remember history".
  • Nhấp vào nút OK để lưu cài đặt của bạn.
Binary (Europe) Ltd

Safari 9.x+

  • Go to the Safari menu and select Preferences.
  • Nhấp vào bảo mật tab.
  • Set "Block cookies:" to Never.
  • Nhấp vào nút X ở góc phải bên trên để lưu và đóng.
Binary (C.R.) S.A.

Opera 38+

  • Go to the Tools menu and select Preferences.
  • Nhấp vào tab Nâng cao.
  • Select Cookies from the left column.
  • Select Accept cookies.
  • Nhấp vào nút OK để lưu và đóng.
Binary Ltd

Microsoft Internet Explorer 11+

  • Go to the Tools menu and select Internet Options.
  • Nhấp vào bảo mật tab.
  • Nhấp vào nút Nâng cao.
  • Select the Override automatic cookie handling.
  • Select Accept for both First-party Cookies and Third-party Cookies.
  • Nhấp vào nút OK để lưu và đóng.
Binary Ltd

Microsoft Edge 12+

  • Go to the More actions menu and select Settings.
  • Under Advanced Settings, click on the View advanced settings button.
  • Under the Cookies section, select the "Don't block cookies" option.
  • Nhấp vào bất cứ nơi nào ngoài trình đơn để lưu và đóng.
